# Password Generator

This is a password generator built using HTML, CSS, and JavaScript. It allows you to generate secure passwords based on customizable settings, such as length and inclusion of various characters like lowercase, uppercase, numbers, symbols, and spaces.

## Features

- **Customizable Password Length**: Set the length of the password from 1 to 30 characters.
- **Character Options**: Choose whether to include lowercase letters, uppercase letters, numbers, and symbols.
- **Exclude Duplicate Characters**: Option to exclude duplicate characters in the generated password.
- **Spaces**: Option to include spaces in the password.
- **Password Strength Indicator**: Visual indication of password strength based on the length.
- **Copy to Clipboard**: Easily copy the generated password with a single click.

## Technologies Used

- **HTML**: Structure and layout of the password generator.
- **CSS**: Styling and layout for a clean, modern design.
- **JavaScript**: Logic to handle password generation, settings, and user interactions.

## Installation

1. Clone this repository to your local machine:

   ```bash
   git clone https://github.com/your-username/password-generator.git
   ```

2. Navigate to the project directory:

   ```bash
   cd password-generator
   ```

3. Open the `index.html` file in your browser to use the password generator.

## Usage

1. Set your desired password length using the slider.
2. Select the character options (lowercase, uppercase, numbers, symbols, spaces).
3. Click the "Generate Password" button to create your password.
4. Copy the generated password using the copy icon next to the input field.
